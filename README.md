# TreClo

## サービス概要
TreCroは、「かんばんスタイル」のタスク管理ツールです。

## 対象ユーザー
チーム開発を行う方で、メンバー間のタスクや進捗状況を共有する方法が確立されていない為に困っている方を対象にしています。

## App URL
https://treclo.net/

## 使用技術
- PHP 8.1
- Laravel 9.11.0
- MySQL 8.0
- Nginx 1.21.6
- Docker/Docker Compose
- AWS
  - VPC
  - EC2
  - ubuntu 22.04
  - ELB
  - Route53
  - ACM
- Vue.js 2.6.12
  - Vue.Draggable 2.24.3
- Bootstrap 5.1.3
